# Brand — Inova Finance AI

## Onde criar / refinar a logo

| Ferramenta | Uso |
|------------|-----|
| **Figma** (recomendado) | Iterar marca, exportar SVG, alinhar com design system |
| **Inkscape** | Vetor gratuito offline |
| **Canva** | Rascunhos rápidos (exportar SVG) |
| **No monorepo** | `packages/ui/src/components/logo.tsx` + `apps/web/public/logo-mark.svg` |

Fluxo: rascunho Figma → export SVG → substituir paths em `Logo` → sidebar/login atualizam automaticamente.

## Paleta v2 (corporativo)

- **Primária:** azul corporativo `#2563EB` / navy `#1A3A5C`
- **Sucesso / financeiro positivo:** emerald `#0D9F6E`
- **Alerta:** gold `#C9A227`
- **Risco:** coral `#E85D4C`

## Símbolo

Quadrado arredondado com gradiente navy→blue, monograma “I” estilizado e traço ascendente (crescimento).

Para importar no Figma: **File → Place image** com `apps/web/public/logo-mark.svg`.
